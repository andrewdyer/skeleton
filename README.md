## Get Started
### Introduction
I created this clutter-free, boilerplate project for myself just to save a little time when developing my next big idea. 

### Installation
Run the following command in terminal to create a new skeleton project:
```text
composer create-project --prefer-dist --stability=dev andrewdyer/skeleton:dev-feat/database project_name
```

### Web Server
Once installed, you can start a localhost web server by running the following command from the project root directory in terminal:
```text
php -S localhost:8888 -t public public/index.php
```
